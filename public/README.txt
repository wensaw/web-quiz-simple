ファイル名を正確に合わせる必要があります。

## 📝 注意事項

- ファイル名は正確に合わせてください
- PNGは透明背景推奨
- 最適化して50KB以下推奨
- ピクセルアート形式
- 原色系（マイクラオリジナル）

## 🎨 画像がない場合

画像がなくても動作します。
絵文字のプレースホルダーが表示されます：

デザイン → 🎨
Figma → 🖌️
コーディング → 💻
正解 → 💎
不正解 → 💥
未回答 → 📦
完了 → ✨

画像をアップロードすれば自動的に表示されます！
